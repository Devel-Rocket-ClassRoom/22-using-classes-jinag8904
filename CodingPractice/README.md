# 코딩 연습 - 클래스 사용하기

교안의 예제 코드를 직접 작성하고 실행해 보세요.

> 여러 과제의 코드를 하나의 `Program.cs`에 작성하면 변수 이름이 겹칠 수 있습니다.
> 과제를 하나씩 작성하고 실행한 뒤, 이전 코드는 주석 처리하거나 삭제하세요.

---

## 과제 1: 클래스 선언과 멤버 호출

`ClassName` 클래스를 선언하고 정적 메서드 `MemberName()`을 정의합니다. 
최상위문에서 이 메서드를 호출하세요.

**실행 결과**

```
클래스의 멤버가 호출되어 실행됩니다.
```

---

## 과제 2: 정적 멤버 호출

`Square` 클래스에 정적 메서드 `GetName()`을 정의하여 "정사각형"을 반환하도록 합니다. 
반환값을 출력하세요.

**실행 결과**

```
정사각형
```

---

## 과제 3: 정적 메서드와 인스턴스 메서드

`MyFirstClass` 클래스에 정적 메서드 `StaticMethod()`와 
인스턴스 메서드 `InstanceMethod()`를 정의합니다. 
정적 메서드는 클래스 이름으로, 인스턴스 메서드는 `new`로 생성한 개체로 호출하세요.

**실행 결과**

```
[1] 정적 메서드
[2] 인스턴스 메서드
```

---

## 과제 4: 개체 생성과 필드 사용

`Person` 클래스에 `Name`과 `Age` 필드, `Introduce()` 메서드를 정의합니다. 
두 개의 개체를 생성하고 각각의 정보를 출력하세요.

**실행 결과**

```
안녕하세요, 홍길동입니다. 25살입니다.
안녕하세요, 김철수입니다. 30살입니다.
```

---

## 과제 5: 참조 타입 확인

`Point` 클래스에 `X`, `Y` 필드를 정의합니다. 한 개체를 다른 변수에 대입한 후,
한쪽에서 값을 변경하면 다른 쪽에도 영향이 미치는지 확인하세요.

**실행 결과**

```
point1.X = 100
point2.X = 100
```

---

## 과제 6: 클래스와 구조체 비교

구조체 `PointStruct`와 클래스 `PointClass`를 각각 정의하고, 
값 복사와 참조 복사의 차이를 확인하세요.

**실행 결과**

```
구조체: ps1.X = 10, ps2.X = 100
클래스: pc1.X = 100, pc2.X = 100
```

## 과제 7: 메서드 정의

`Calculator` 클래스에 `Add()`와 `Multiply()` 메서드를 정의합니다. 
`Multiply`는 식 본문 메서드(=>)로 작성하세요.

**실행 결과**

```
3 + 5 = 8
3 * 5 = 15
```

## 과제 8: 생성자 오버로딩

`Player` 클래스에 기본 생성자와 매개변수가 있는 생성자를 정의합니다. 
두 가지 방법으로 개체를 생성하고 정보를 출력하세요.

이름: 용사, 체력: 150

## 과제 12: new 키워드와 개체 생성

## 과제 14: Random 클래스 기본 사용

`Random` 클래스의 인스턴스를 생성하고 `Next()`, `Next(max)`, 
`Next(min, max)`, `NextDouble()` 메서드를 사용하여 다양한 난수를 생성하세요.

**실행 결과 (예시 - 실행할 때마다 값이 달라짐)**

```
임의의 정수: 1234567890
0 ~ 4: 3
1 ~ 9: 7
0.0 ~ 1.0: 0.4521
```

## 과제 15: 주사위 프로그램

`Random` 클래스를 사용하여 주사위를 5번 굴리는 프로그램을 작성하세요. 
`for` 반복문과 `Next(1, 7)`을 사용합니다.

**실행 결과 (예시 - 실행할 때마다 값이 달라짐)**

```
주사위를 5번 굴립니다:
  1번째: 4
  2번째: 2
  3번째: 6
  4번째: 1
  5번째: 3
```
